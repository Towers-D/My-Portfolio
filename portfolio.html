Grapple Run

This is a game I worked on during my final year of my BSc. I made the game in Unity using assets off of the Unity asset store.
The game is a simple 2D platformer, where the player traverses through the levels until the run off screen, however, the game is designed to only be traversable via the grapplehook the player starts with. The player uses the grapplehook to swing over obstacles, lower themselves down and pull themselves up, as well as a tool for defeating enemies.

The game was made to a deadline, and is currently in an unfinished state. I plan to finish the game in the future, which will include:

  • Reorganising the code in the scripts
  • Changing how the rope interacts with the world (Making it bend around objects, and behave like a rope
  • More levels with more difficult puzzles
  • Finished animations
